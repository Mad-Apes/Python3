# 除法

## 真除法 X / Y
```
  Python2.6 之前的版本，除法，对于整数操作来说结果会省去小数部分，对于浮点数操作结果会保留小数部分。
  但是在 Python3 中，无论对于任何类型都会保留小数部分即为真除法。
  DEMO:
    print(0/1)        # 结果为 0.0

  添加真除法的目的是为了解决之前传统的除法结果依赖于操作数类型（因此，其结果在Python这样的动态类型语言中很难预料）。【摘自 Python学习手册第四版 第5张 除法章节】
```

## floor 除法【截断除法】 X // Y
```
  含义是把结果向下阶段到它的底层，既真正结果之下最近的整数。其结果是向下舍入
  这个除法操作符不会考虑操作对象的类型，总会省略掉结果的小数部分，剩下最小的能整出的整除部分。
  DEMO:
    print(1//3)         # 结果为 0
    print(2.0//3)         # 结果为 0.0
    print(11//3)        # 结果为 3
    print(-11//3)       # 结果为 -4
    print(-11.0//3)     # 结果为 -4.0
```

## 总结
```
  在 Python3 中
  / 总是执行真除法，即不论备操作数是什么类型，结果总会保留小数部分，即返回一个浮点数。
  // 执行 floor 除法，针对整数操作数，返回一个整数类型，如果任何一个操作数为浮点数类型，则返回一个浮点数。
```
