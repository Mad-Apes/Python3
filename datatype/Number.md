 ## 数字类型
 ```
	注意：
		Python3 中的数字类型存储数值，该类型是不可变的。这就意味着改变数字类型的值得话，必须重新分配内存空间。
 ```
 ## 整数 int
	### 整数包括正整数和负整数。
	DEMO:
	````
		var_num_1 = 100
		
		var_num_2 = -3
	````
	Python 允许整数具有无限精度，只要内存空间允许，整数可以是无限长度的数字。
	
	### 浮点数 float 带有小数部分的数字
	DEMO:
	```
		var_num_1 = 3.14
		var_num_2 = 3.14e-10
	```
	
	### 复数 complex
	DEMO:
	```
		var_num_1 = 3 + 4j
		var_num_2 = 3 + 4J
	```
